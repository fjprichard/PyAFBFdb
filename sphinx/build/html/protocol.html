Class protocol

class afbfdb.Protocol.protocol(rep='SimulationSet_001', M=2, smode_cst='unifmin', Hmin=0.01, Hmax=0.99, smode_int='nonunif', dint=0.001, K=500, N=64)

Bases: object

This class enables to manage and build database.

Parameters

  • rep (str) – Path to the data directory.

  • M (int) – Number of parameters for the Hurst and topothesy functions.

  • smode_cst (str) – Mode of simulation of the Hurst and topothesy function.

  • Hmin (float) – Minimal value of the Hurst index in \((0, 1)\).

  • Hmax (float) – Maximal value of the Hurst index in \((0, 1)\).

  • smode_int (str) – Mode of simulation of the intervals of the Hurst function.

  • dint (float) – Minimal interval lenght of the Hurst function steps.

  • K (int) – Number of turning bands.

  • N (int) – image size.

ExportData(n_start=0, n_end=None)

Export the data.

Parameters

  • n_start (int) – first example.

  • n_end (int) – last example.

Returns

(images, features)

Return type

(ndarrays, ndarrays)

Note

  • images is an array of size N x N x (n_end - n_start + 1): images[j, :, :] is the image of the (n_start + j)th example.

  • features, an array of size N x N x (n_end - n_start + 1): features[j, :] are the features of the (n_start +j)th examples.

    • features[j, 0] is the Hurst index.

    • features[j, 1] is the lenght of the Hurst argmin set.

    • features[j, 2] is the center of the Hurst argmin set.
